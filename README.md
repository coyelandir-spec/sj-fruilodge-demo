# S&J FRUI'LODGE — Démo du site

Démo statique (HTML/CSS/JS) du futur site vitrine et commercial de **S&J FRUI'LODGE**
(Voyages & Maison d'hôte) à Antsiranana — Diego-Suarez, Madagascar.

## ⚠️ C'est une démo

- Sert à valider le **design, l'ambiance et la structure**.
- Les **réservations** et **paiements** ne sont **pas actifs** (formulaires de démonstration).
- Les **textes, tarifs et photos** sont des **placeholders** entre crochets `[ ... ]`,
  à remplacer avec le PDF et les informations de l'entreprise.

## Pages

| Fichier | Page |
|---|---|
| `index.html` | Accueil |
| `chambres.html` | Nos Chambres & Suites |
| `vehicules.html` | Nos Véhicules |
| `excursions.html` | Excursions & Activités |
| `reservation.html` | Réservation & Paiement |
| `contact.html` | Contact |

## Voir la démo

- **En ligne** : voir le lien GitHub Pages (ajouté après déploiement).
- **En local** : double-cliquer sur `index.html`.
